## 무엇을 / 왜
현장 외국인 노동자가 "작업온" 앱 전체 UI를 자기 언어로 사용하도록 6개 언어 다국어화.
지원 언어(웹과 동일): **한국어(기본) + 중국어 간체·러시아어·베트남어·네팔어·영어**.

## 어떻게 (설계)
- **flutter_localizations + intl ARB 방식**: `l10n.yaml`(arb-dir `lib/l10n`, 템플릿 `app_ko.arb`,
  `nullable-getter: false`) + `flutter gen-l10n` 생성 코드(`AppLocalizations`).
  - `lib/l10n/app_{ko,zh,ru,vi,ne,en}.arb` — **ko 가 기준(template)**, 총 **417키 × 6언어**(키 패리티 스크립트 검증, 결손 0).
  - `lib/l10n/l10n_ext.dart` — `context.l`(AppLocalizations)·`context.lang`(언어코드) 단축 확장.
  - 웹 사전과 겹치는 키(종이 확인서 라벨 paper*·share*·status* 등)는 **웹 문구 그대로 재사용**(번역 일관성).
- **언어 선택**: 더보기(설정) → `언어` 타일 → 바텀시트(시스템 따름/한국어/中文/Русский/Tiếng Việt/नेपाली/English,
  자국어 표기 고정). `lib/providers/locale.dart`(`LocaleController`, shared_preferences `workon_lang` 저장).
  기본은 **시스템 로케일 → 미지원이면 ko**(`localeResolutionCallback`). `main.dart` 에 delegates/supportedLocales 배선.
- **날짜·금액(웹과 동일 규칙)**: `lib/core/format.dart` 에 로케일 대응 함수 추가 —
  `formatMoney(v,lang)`(ko `1,234원`/그 외 `₩1,234`, 원화 고정), `formatGrouped`, `fmtDate`/`fmtShortDate`/
  `fmtFullDate`/`fmtMonth`(Intl 로케일 요일·월), `fmtAmpm`. `LANG_LOCALE` 매핑(ko_KR/zh_CN/ru_RU/vi_VN/ne_NP/en_US) 웹과 동일.
  기존 한국어 전용 함수(formatWonUnit 등)는 홈택스 텍스트 등 한국어 고정 경로에만 잔존.
- **전 화면 문자열 추출**: 로그인/온보딩, 홈, 캘린더(요일 헤더 `DateFormat.E` 로케일), 확인서 작성
  (단가유형 세그먼트·공수 스텝퍼·검증 문구·초안/자동보존), 확인서 상세/공유, 장부/회사상세(D-day
  `수금 D-3`·`기한 지남`·`입금완료` 라벨 번역, D-3 표기는 유지), 세금계산서(+사업자정보), 서류 지갑
  (업로드/상세/마스킹 편집기/묶음 공유/내 공유/장비), 사업장 모드 전체(생성/수신함/앱내서명/정산/작업자/
  작업지시/안전), 받은 작업(컨디션 체크), 알림, 더보기/설정, 초안 목록, ErrorRetry·스낵바·다이얼로그 공통.
- **금액 항목 라벨은 type 기준 번역**(웹 규칙 동일): BASE(rateType→기본(일당)/공수 등)·OVERTIME/EARLY/NIGHT/ALLNIGHT.
  OTHER 사용자 라벨은 데이터로 유지. `amount_calc.dart` 내부 맵(서버 동일 로직)은 한국어 유지, **표시 시점**에만 번역.
- **공수(gongsu) 개념어**: zh `工数`, ru `гонсу`, en/vi/ne 로마자 `gongsu`(+설명형 라벨 `단가 (1 gongsu = 1일)` 식 병기).
- **번역하지 않음(정책)**: 사용자 데이터(현장·회사·이름·서류유형 값), **홈택스 복사 텍스트
  (`tax_invoice_text.dart` 한국어 고정 — 정책 확정)**, 확인서 PDF(백엔드 소관), 브랜드 `작업온`(스탬프는 WORKON 로마자).
- **긴 문자열 레이아웃 대응**(러시아어 등): 실측에서 발견된 오버플로 수정 —
  장부 월 헤더(FittedBox scaleDown + Expanded), 작성 폼 상대 칩(Row→Wrap), 홈 요약 금액·폼 날짜/시간 칩
  (FittedBox scaleDown, **금액·시간은 잘림 대신 축소**).

## 남은 항목 (위젯 배치 등 후속에 넘김)
- **ApiException 폴백 메시지 한국어**(`core/api_client.dart` `서버에 연결할 수 없습니다` 등): 화면측 래퍼
  (`저장 실패: {message}`)는 번역됐으나 서버/클라이언트 원문 메시지는 한국어. 에러 **코드→로컬라이즈 문구 매핑
  레이어**(NETWORK/UNAUTHORIZED 등 대표 코드만이라도) 후속 권장 — 서버 message 자체가 한국어라 서버측 코드화와 함께.
- **서류 유형 표시 번역**: 유형 값(자격증/보험증권…)은 서버 데이터로 미번역(웹 정책 동일). UI 표시용
  알려진 유형 매핑(값은 한국어 유지, 표시만 번역)은 별도 배치로.
- 데이터 폴백 문자열(`models.dart` `(미지정)`/`서류`/`장비`, `draft_store.dart` `상대 미지정`) — 파싱 시점이라
  로케일 접근 불가, 표시 시점 치환 여지.
- 캘린더 압축 금액 단위(ko/zh 만·万, 그 외 k/тыс/हजार) 원어민 검수 및 라벨 확대.
- 홈 요약 금액 카드가 좁은 폭에서 scaleDown 으로 작아짐 — 위젯 배치 개선 시 카드 높이/랩 재설계 여지.
- 확인서 **PDF 이중언어 옵션**(백엔드 소관, 웹 worklog 이월 항목 그대로), 알림(푸시/알림톡) 본문 다국어(백엔드 소관).
- 2차 확장 언어(크메르·인니·태국·미얀마·몽골)는 ARB 파일 추가 + `supportedLangs`/`langNative` 항목 추가만으로 확장.
