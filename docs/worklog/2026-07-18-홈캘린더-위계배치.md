## 무엇을 / 왜 / 어떻게

### 1. 홈 "내 돈" 단일 히어로 (`app/lib/features/home/home_screen.dart`)
- 인사말 아래 최상단에 **"이번 달 받을 돈 650,000원"** 단일 대형 숫자(40px, 미수색, tabular)를 히어로로 세움.
- 받은 돈·일한 날은 히어로 하단 보조 스탯 한 줄로 흡수 → 기존 "이번 달 요약" 2분할 카드(`_SummaryCard`/`_SumCell`) 제거(중복 섹션 없앰).
- 화면 순서: 인사 → 히어로 → CTA(확인서 쓰기) → 오늘 일정 → 확인 필요 배너.
- 신규 위젯 `_MonthHero`/`_HeroStat`. 데이터는 기존 `ledgerSummaryProvider` 재사용(새 API 없음).

### 2. 캘린더 장부화 (`app/lib/features/calendar/calendar_screen.dart`)
- 월 그리드에서 날짜 탭 시 **모달 바텀시트(`_openDay`) 제거 → 그리드 아래 인라인 장부 패널(`_DayLedger`) 펼침**. 같은 날 재탭 = 접기(`_selectedDayProvider`).
- 패널 행(`_DayLedgerRow`): 현장명 + 큰 금액(미수색) + 상태 배지(`_StatusChip`, 백엔드 statusLabel·SIGNED/SENT/DRAFT 색 구분) + 상대·시간. 탭 → 확인서 상세.
- 월 상단 합계를 **"받을 돈" + 미수색 큰 숫자(26px)**로 홈 히어로와 동일 서식화. 미선택 시 안내 문구(`calTapDayHint`).
- 선택 셀 하이라이트(주황 채움+테두리) 추가. 날짜셀에 `ValueKey('cal-day-YYYY-MM-DD')` 부여(테스트용).
- 월 이동 시 선택 날짜 초기화.

### 3. 홈 만료 문구 중복 버그
- 기존 `homeDocExpiry`="{type} 만료 {dday}"에 만료 시 `docExpired`("만료됨")를 주입 → "자격증 만료 만료됨" 중복.
- 템플릿을 `"{type} {status}"`로 바꾸고, 임박은 신규 `homeDocExpiryDue`("만료 {dday}"), 만료는 `docExpired`("만료됨") 사용. 결과: "자격증 만료됨" / "자격증 만료 D-7".
