## 개선 항목 (P1 8건)

1. **영문 enum 노출 전량 제거** — `bizSignDone`·`taxEmptySubtitle` 두 문자열의 `(SIGNED)` 제거(ko·en·zh·vi·ru·ne 6언어 일괄). 잔존 확인: 지갑/토스트/세금 빈 화면에서 영문 enum 0. (스탬프 `서 명 완 료 · WORKON` 등은 의도된 브랜드 각인이라 유지.)
2. **서류 묶음 보내기 발견성** — 지갑 상단에 상시 노출 **`선택해서 보내기`** 버튼(체크리스트 아이콘, 오렌지 아웃라인) 추가. 탭 → 선택 모드(카드 라디오 체크 + 하단 전송바). 기존 롱프레스 진입도 유지. (`wallet_screen.dart`)
3. **죽은 공간 화면 밀도** — 정산(사업장) 화면 상단에 **`이번 달 지급할 돈` 합계 카드**(30px 큰 숫자·건수) 추가해 핵심 숫자를 1스크린 상단에 배치. 확인서 상세는 이미 하단 액션바(전송·문자·전화) 보유 확인. (`settlement_screen.dart`)
4. **장부 항목 버튼 정리** — 회사 상세 항목을 **주 행동 `[입금 기록]` 1개(아웃라인 버튼) + `⋯` 오버플로**로 정리. `⋯` → `이 건 관리` 바텀시트(자동 수금 안내 토글·지금 안내 보내기·발송 이력). 완납/파생 행은 액션 숨김 유지. (`company_detail_screen.dart`)
5. **D-day 표기 통일 + 공용화** — `ddayUnified(l, dday)` 단일 규칙 함수 신설: 임박 `D-N` · 당일 `D-day` · 지남 `+N일`(부호 혼용 `D+N` 폐기). 만료/완납은 의미 라벨(`만료됨`·`입금완료`) 유지. 장부(`ddayText`)·서류(`DocStatusBadge`)·홈·지갑 배너 전 화면이 이 함수를 경유. l10n `ddayOverdue` 신설(6언어). (`widgets/common.dart`, `doc_ui.dart`, `home_screen.dart`)
6. **아이콘 스트로크 통일** — 감사 지적 필드 필-라운드 아이콘 → 아웃라인 계열로: `share_rounded→share_outlined`, `send_rounded→send_outlined`, `link_rounded→link`(전 features/widgets 일괄). 아웃라인 2px 언어로 수렴.
7. **홈 상시 CTA** — 홈 상단(초안 배너 아래)에 상황 무관 상시 노출 **`확인서 쓰기` 56px 주 CTA** 추가(빈 상태/데이터 유무 무관). (`home_screen.dart`)
8. **서명 표시 = 도장(印) 박스** — `SignatureSeal` 위젯 신설: 붉은 도장 원(서명자명 각인·−0.1rad 기울임) + `{name} 님 서명 완료` + 서명 일시. 확인서 상세·수신함(biz) 서명 완료 시 텍스트 배지 대신 도장 박스 렌더. `Confirmation` 모델에 `signedAt` 파싱 추가. 
   - **한계(정직 고지)**: 서명 **래스터 이미지**는 백엔드가 JSON DTO로 노출하지 않는다(`signImagePath` 내부 전용, 이미지 서빙 엔드포인트 없음 — 확인 완료). PDF 생성에만 사용. `app/ 만 수정` 제약상 앱에서 원본 획 이미지를 받을 수 없어, **서명자명 각인 도장 박스**로 시그니처를 실현. 원본 획 렌더는 백엔드 DTO 1필드 추가(범위 밖)가 선행돼야 함.
