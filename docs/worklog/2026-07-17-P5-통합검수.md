## 2. 세액 계산 독립 재검증 (소스 무관 국세 기준 손계산 → 순수함수/e2e/실서버 3중 대조 일치)
독립 파이썬 손계산 + 실서버 curl(pg 5438/api 3060 라이브 시드)로 대조. **전부 일치.**

**일용근로 산식**(코드): `1일세액 = floor10((일급−150,000)×6%×45%)`, `<1,000 → 0(소액부징수)`, 일수 합산 후 재 floor10, 지방세 = `floor10(소득세×10%)`.

- **15만 이하** — 일급 150,000 → 과세표준 0 → **0원**. (실측 이*호 15만/1일 계열 및 별도 케이스)
- **소액부징수 경계(999 vs 1,000) — crisp·정확**:
  - 일급 **187,000**(과세표준 37,000, 실세액 37,000×2.7%=**999** < 1,000) → **0원**(소액부징수). 실측 김*수 역삼 187,000×1일.
  - 일급 **187,100**(과세표준 37,100, 실세액 **1,001.7** → floor10 1,000 ≥ 1,000) → **1,000원**. 실측 김*수 판교 GONGSU 187,100.
  - 김*수 지급총액 374,100(위 2건) → 일용 소득세 0+1,000=**1,000**·지방 100·합 1,100·차인 373,000. (수학적으로 `floor10(x)<1000 ⇔ x<1000` 이라 경계 무결)
- **다일수** — 일급 200,000×3일 → 1일 (200,000−150,000)×2.7%=**1,350**·×3=**4,050**·지방 400·합 4,450·차인 595,550. 실측 이*호 60만/3일.
- **3.3% 지방소득세 절사 일관성(원단위·10원단위)**:
  - 600,000 → 소득세 floor10(18,000)=18,000·지방 floor10(1,800)=1,800·차인 580,200.
  - 374,100 → 소득세 floor10(11,223)=**11,220**·지방 floor10(1,122)=**1,120** — **원단위(10원 미만) 절사** 소득세·지방세 양쪽 동일 규칙(floor10) 적용 일관. (단위테스트 33,333→990 도 동일 floor10.)
- **정합**: site-costs **총계 = 확인서 합**(실측 1,274,100 = 187,000+600,000+187,100+300,000, 소계=행합 invariant 내장). wage-statement **지급총액 = payments 합**(실측 974,100 = 187,000+600,000+187,100, 판교2 30만 미지급분 제외).
