# 프로젝트 개요 (PROJECT)

## 비전
업종에 상관없이 **작업지시자(회사/현장)와 작업자(기사/기술자)** 가 서로를 빠르게 찾고,
서류를 간편하게 주고받고, 일한 내역(작업확인서)을 공유·확인하고, 정산과 수금까지
한곳에서 관리하는 **무료 앱**.

궁극 목표: 작업자는 "할 일 + 받을 돈 + 서류 만료"를 장부처럼 관리하고,
회사는 "작업 확인 + 정산"을 신뢰 기반으로 처리한다.

## 핵심 요구사항 (2026-07-11 사용자 설명 기준)

### 1. 서류 업로드·전달
- 장비/이력 관련 서류를 업로드 (어떤 이미지 포맷이든, PDF든 허용)
- 전송 시 자동으로 **PDF로 변환**하여 전달
- 링크 공유, 메일, 카카오톡 등으로 빠르게 상대에게 전달

### 2. 지시자–작업자 연동 (매칭/검색)
- 작업지시 회사는 **작업자를 전화번호로 검색**해서 바로 일을 시킬 수 있음
- 작업자는 **회사 이름 또는 코드번호로 검색**해서 연동
- 서로 연결되면 즉시 작업 지시/수락 가능

### 3. 작업자 기능 (스케줄·작업확인서·수금 장부)
- 스케줄 관리로 일한 내역을 쉽게 정리
- 매일 **작업확인서**를 작성하고 작업 회사와 공유
- 회사가 미가입인 경우: 작업확인서를 **파일/링크로 카카오톡 등 외부 전송** 가능
- 장부: 언제/어디서/누구의 일로/얼마를 받을지 + **수입 예정일** 관리
- 받은 돈 / 받을 돈 명확히 구분
- **알림**: 수금일 알림, 작업 예약 확인 알림

### 4. 지시자 기능 (작업 확인·정산)
- 작업자의 **일 시작 + 완료 체크** (필요시 사진 첨부) 확인
- 제출된 작업확인서 확인 → **정산 관리**

### 5. 만료 관리 (신뢰 기반)
- 작업자 서류 등록 시: 면허, 사업자, 차량 검사일 등 **안전 관련 만료일 관리 자동화**
- 만료일은 수동 등록, 앱이 알림으로 관리
- 서류 유효 상태를 바탕으로 상호 신뢰 작업 가능

### 6. 안전·자동화 (2026-07-11 추가)
- **날씨/폭염 자동 안내**: 아무리 작은 회사라도 폭염 경고, 휴식 안내를 알림톡/푸시로 자동 발송
- **부가세(세금계산서) 자동 발행**: 정산 확정 시 세금계산서까지 자동으로 이어지게
- **설계 철학: "복잡성보다 자동화"** — 편하고 꼭 필요한 것만. 기능 수가 아니라 자동화 수준으로 승부

### 7. 안전관리 증빙 (2026-07-11 추가 — 지시자 측 핵심 가치)
- 앱이 수행한 모든 안전 행위를 **자동으로 기록**하여 회사의 안전관리 의무 이행 증거로 활용:
  - 폭염 경고/휴식 안내 발송 기록 + 작업자 수신·확인 기록
  - 작업자 서류(면허/보험/검사) 유효성 확인 기록
  - 건강 체크(컨디션 확인) 기록
- 기록을 **"안전관리 이행 리포트" PDF**로 내보내기 → 중대재해처벌법/산업안전보건법 대응 증빙
- 마케팅 포지셔닝: "무료로 쓰는 안전관리 증빙 시스템" — 소사업장의 가입 명분

### 8. 디자인 v1 피드백 (2026-07-11 사용자 확정)
- **서류 보내기**: 서류를 개별 선택(체크)해서 골라 보낼 수 있어야 함. **주민번호 마스킹 필수**.
  면허·사업자 진위확인은 SKEP(다른 프로그램)에서 구현한 방법과 API 키를 참고해 구현.
- **일정**: 달력 보기 옵션 — 월간/주간 한눈에 보기 전환. 사용자 편리성 최우선.
- **사업주 측**: 사업주도 같은 앱 설치 → 빠른 연동 → 연동되면 작업확인서 공유 + 정산까지 연동.
  작업 지시 + 안전 관련 자동 알림 구현 필수.
- 개발 지시: **모든 과정을 순차 진행, 단계별 검수 필수** ("모두 진행해")
